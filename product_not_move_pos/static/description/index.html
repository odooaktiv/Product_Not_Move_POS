Product not moving Report

This reports will display all POS products which are not\ sold in selected period. User can also filter products by warehouse.

Prerequisites before Installation

No prerequisites required before installation.

Configurations

No Configurations are required.

Step-By-Step Guidelines

Product not moving POS Report module provides easy to print report.

With this module users can get a report which consists of a list of products which haven’t recognized any sale from the warehouse.

There is one wizard open once click on Product Not Move menu to print report and Enter the start and end date for particular warehouse which you want to get a report of not moved product.

Once you click on print button user can get report with information like Here the duration field is the one which stores the difference between last sale date and start date. Suppose the start date is 04/06/2020 and the last sale date is 29/05/2020 then the duration gap will be of 6 days. Here’s the example of Product Not Move Report
